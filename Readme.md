# Gourmet - Mealie for iOS

A small iOS app for your self-hosted Mealie server

## About the Project

Mealie is a self hosted recipe manager web app. Gourmet / Mealie for iOS is a small iOS app that makes it easy to cook with the recipes you have on your Mealie instance

You can get it from the [AppStore](https://apps.apple.com/us/app/gourmet-for-mealie/id6749816018)
